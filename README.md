# Máster en IA — AIC · Prácticas

Material de los **casos prácticos** del módulo *IA Aplicada al Vehículo Autónomo* del Máster en IA de **AIC**.

Repositorio **público**: puedes descargar todo lo necesario para realizar las prácticas.

## Casos prácticos disponibles

| Caso | Carpeta | Tema |
|------|---------|------|
| **CP1** | [`CP1-carriles/`](CP1-carriles/) | Detección de carriles: clásico vs deep learning |

*(Se irán añadiendo CP2–CP5.)*

## Cómo empezar

1. Descarga o clona este repositorio.
2. Entra en la carpeta del caso práctico (p. ej. `CP1-carriles/`).
3. Abre el **dosier del alumno** (`rendered/CP1-dosier-alumno.pdf`) y sigue las instrucciones paso a paso.

Las imágenes y los modelos de gran tamaño se descargan mediante el script `scripts/download_assets.py` de cada caso, desde las *releases* de este mismo repositorio.
